# Original images
This folder contains 240 original images captured by Azure Kinect, RealSense D435, and ZED 2i, 80 images per camera, and the images naming format is from 000001 to 000080. 

# Augmented images

After dataset augmentation, the number of images per camera in the training set increases from 64 images to 1152 images. Therefore, this folder contains 3456 images used to train the Yolov5x network, and the image naming format is from 000001 to 001152.
